Co znaczy skrót ID? Identyfikatory
Jaki jest cel stosowania ID? Identyfikator ID pozwala na jednoznaczne określenie konkretnego elementu na stronie. Jest przydatny, gdy chcemy nadać unikalny styl lub odwołać się do danego elementu w JavaScript.
Jakie jest znaczenie znaku #? Znak # jest używany w CSS do oznaczania identyfikatora (ID) elementu, co pozwala na jego jednoznaczne stylowanie.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie identyfikatora w CSS polega na użyciu selektora, który składa się z symbolu # oraz nazwy identyfikatora. Przykład: #nazwa_id. W ten sposób przypisujemy odpowiednie style do elementu z określonym ID w HTML.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  • Różnica między id a class: Identyfikator (ID) może wystąpić tylko raz w całym dokumencie, natomiast klasę (class) można przypisać do wielu elementów.
  • Choć identyfikatory powinny być unikalne, można je używać wielokrotnie, co jest błędem, ale nie wpływa na funkcjonowanie strony.
  • Elementy oznaczone ID muszą być unikatowe, czyli na jednej stronie może występować tylko jeden element z danym ID. Jeśli nie mamy pewności, użyjemy 'class', która może być przypisana do wielu elementów.
